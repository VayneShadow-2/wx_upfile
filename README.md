### wxDownLoad

微信小程序中 word，doc，txt等文件上传与下载
本案例支持小程序中文件上传与下载

##### 文件上传两种方式 
- [x] 上传当前小程序中下载过的文件
- [x] 利用wx.chooseMessageFile  调用微信会话页面 选择要上传的文件

**项目启动**  

> wx_static 目录为小程序项目

> node 目录为node项目，
- upload   提供文件下载的静态目录
- static   接收文件上传的目录


1. wx_static/config  更改ip地址

2. cd/到node 

3. cnpm install （安装依赖）  

4. node app.js （启动node服务）
